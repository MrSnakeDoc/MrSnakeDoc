## 🧠 About Me

I'm a backend developer with a solid foundation in Node.js and TypeScript, currently focusing on building systems in **Go** for their performance, clarity, and maintainability.

My daily work revolves around **self-hosted architectures**, automation, infrastructure observability, and clean backend service design.

My approach is pragmatic: I build systems, test them in production-like conditions inside my homelab, and iterate. With growing expertise in DevOps, I continue to expand toward **Kubernetes**, **infrastructure as code**, and **cloud-native operations**.

## 🏗️ Homelab & Infrastructure

This space is where I build, test, and operate most of my tools and services:

- 🐳 Docker-based microservice architecture on Ubuntu
- ⚡ Traefik as reverse proxy with full Let's Encrypt automation
- 🔐 Cloudflare Tunnels (Zero Trust setup)
- 📈 Monitoring stack: Prometheus, Grafana, Node Exporter, cAdvisor
- 🛡️ AdGuardHome for local DNS and content filtering
- 💾 Synology NAS + Router for media, backup, VPN, and DNS resolution
- 🧰 Hosted services: Gitea, Drone CI, Uptime Kuma, Gotify, Mealie, Outline, Homepage
- 🧱 Everything runs on Proxmox, entirely containerized
- ➕ And many more.

> *Kubernetes (K3s) is currently in the migration plan.*

## 📦 Projects

### ✅ Released

- [**Keg**](https://github.com/MrSnakeDoc/keg):  
  CLI tool to bootstrap and manage a local development environment with Go. Automates ZSH, Homebrew, and packages setup.

- [**homelab-blueprint**](https://github.com/MrSnakeDoc/homelab-blueprint):
